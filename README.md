# FileServer

## Run server
```
go run .
```

## Run client
```
telnet localhost 5000
```

### Commands available in client

#### Set a user
```
/user USER
```

#### Get all channels
```
/channels
```

#### Join to channel X
```
/join CHANNEL
```

#### Send a message to channel X /
```
/msg MESSAGE
```

#### Send a file to channel X /
```
/file FILE
```

#### Quit
```
/quit
```
